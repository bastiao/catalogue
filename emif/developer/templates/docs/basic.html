The plugin structure

Below follows some basic directions on how to develop a plugin, and the absolute essentials for the to work on the EMIF Catalogue plugin platform.

Basic Example

At heart, a plugin for EMIF Catalogue is very simple. Only needed two mandatory things in it's structure, a confs dictionary and a plugin function, that receives a self parameter.

Below follows the typical, hello_word plugin.


confs = {
    'name': 'Test plugin'
};
plugin = function(self){
    self.html('Hello world');
    self.refresh();
};

The confs dictionary

The confs dictionary is the place that allows specification of the widget configurations. Settings such as the display name, the widget icon, and the preferential position and size are defined in this array. Below follows a list of all the parameters available.

name default: "New Plugin"

A string that defines the display name for the plugin.

icon default: undefined

A string that defines the icon for the plugin.

extracss default: []

An array of strings, that defines some extra css urls that should be loaded to be able to execute the plugin.

extralibs default: []

An array of strings, that defines some extra javascript library urls that should be loaded to be able to execute the plugin.

Global widget specific configurations:

width default: 2

Defines the preferential width for the plugin.

height default: 1

Defines the preferential height for the plugin.

x default: 1

Defines the preferential x coordinate for the plugin.

y default: 1

Defines the preferential y coordinate for the plugin.

The plugin function

The plugin function is the heart of the plugin. Is on this function that the main heart of the application goes. This plugin function has access to a series of objects (detailed on the section "Available objects").

Besides that, it is possible to use jquery, bootstrap 2.3.2 styling and font-awesome 4.3 icons.

If there is a need to use other external libraries, not included in the list, they can be included through the confs settings "extra_libs", already aborded above.
