Get Token and Fingerprint IDs

First, you need access to the information of the API Key (Token) and the fingerprint IDs associated with your account.

To get that information, you should go to Workspace / My Databases and click on API Info button.

It will be displayed all information that you need to use API Services.

Insert/edit values to a fingerprint

The parameteres to insert values in one fingerprint are:

  • Url: http://bioinformatics.pt/emif/api/metadata
  • Method: POST or PUT
  • HTTP Header: "Authentication: Token <your token>"
  • JSON structure: 
    {
 "fingerprintID":"<your ID>",
 "values":
    {
        "field1":"value1",
        "field2":"value2",
        "field3":"value4",
        ...
    }
}

Note: If the field does not exist, it will be created, otherwise the value will be replaced.

Example
curl -X POST -H "Content-Type: application/json" -H "Authorization: Token <your token>" -d "{\"fingerprintID\":"<your ID>",\"field1\":\"value1\"}" http://bioinformatics.pt/emif/api/metadata

Get values of one fingerprint or all

The parameteres to get values are:

  • Url: http://bioinformatics.pt/emif/api/search
  • Method: GET
  • HTTP Header: "Authentication: Token <your token>"
  • Data structure: 
    data: {
     "fingerprintID":"<your fingerprint ID> "
}
    or 
    data: {
     "fingerprintID":"all"
}

Note: The output will be the structure of fingerprint(s) (fields and values).
